Kod stranice "Filmovi" se ne prikazuje cijeli sadržaj na pregledniku kako se učitava lokalna JSON datoteka te CORS iz sigurnosnih razloga ne dozvoljava njegovo učitavanje, stoga je snimljena video prezentacija njenog prikaza i rada. 

Ukoliko želite sami isprobati rad projekta, moguće je u Visual Studio Code-u preuzeti ekstenziju "Live Server" (prvu koju izbaci), zatim je potrebno otvoriti mapu projekta te desni klik na bilo koju html datoteku i odabrati opciju "Open with Live Server". 
Tek tada ćete moći vidjeti cijeli sadržaj stranice "Filmovi".
